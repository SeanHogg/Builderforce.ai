# Basis Payload Specification - Changelog

This document tracks changes to the Basis Payload JSON schema and specification.

## [1.0.0] - 2025-08-24

### Added
- Initial release of the Basis Payload specification
- JSON Schema (Draft 2020-12) for programmatic validation
- full specification with all field definitions, constraints, and examples
- Canonical example payload demonstrating all features
- Command-line validation utility

### Features
- Schema versioning support (SemVer)
- Claim blocks with confidence modeling
- Evidence referencing with provenance tracking
- Reasoning chain (optional)
- Uncertainty/caveats summary
- Operational context metadata
- Extension mechanism for future-proofing
