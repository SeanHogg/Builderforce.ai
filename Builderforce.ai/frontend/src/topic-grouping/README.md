# Chat Topic Grouping Feature

This feature automatically identifies and groups chats by their thematic content to reduce visual clutter and improve information retrieval.

## Overview

The system analyzes chat messages, titles, and participants to infer dominant topics and groups chats accordingly. Users can then:
- Rename automatically generated topic groups
- Merge multiple groups into a new group
- Move chats between groups (drag-and-drop or context menu)
- Ungroup chats to make them appear as standalone items

## Core Components

### Types (`types/chat.ts`)
- `Chat`: Represents a complete chat object with messages and metadata
- `ChatTopicGroup`: Stores information about a topic group
- `TopicDetectionResult`: Result of topic detection with confidence scores
- `AutoGeneratedGroup`: Auto-generated group configuration (emoji, color, etc.)

### Topic Detection (`utils/topicDetection.ts`)
- **Keyword Patterns**: Pre-defined regex patterns for common topics (PRD, PWA, Agent Creation, etc.)
- **Detection Functions**:
  - `detectTopic()`: Analyzes chat content and returns topic with confidence
  - `categorizeChat()`: Classifies chats into topic categories
  - `identifyPrimaryTopic()`: Gets the primary topic for a chat
  - `groupChatsByTopic()`: Groups multiple chats by their topics
  - `analyzeChats()`: Provides comprehensive analysis of grouped chats
  - `generateGroupedAnalysis()`: PRD-oriented grouping helper

### Group Management (`utils/groupManagement.ts`)
- **Custom Group Operations**:
  - `createCustomGroup()`: Create a user-managed group
  - `renameGroup()`: Rename a custom group
  - `mergeGroups()`: Merge multiple groups into one
  - `moveChat()`: Move a chat between groups
  - `ungroupChat()`: Remove a chat from all groups
- **Helpers**: Sort by count, get top groups, validate names

### API Client (`lib/topicGroupApi.ts`)
- **Client-side API calls** for:
  - Topic detection
  - Group creation/update/deletion
  - Chat reassignment
  - State persistence
- **Local Fallback**: Generates analysis when API is unavailable

## Usage Example

```typescript
import { detectTopic, groupChatsByTopic } from '@/utils/topicDetection';
import { mergeGroups, ungroupChat } from '@/utils/groupManagement';
import { createCustomGroup } from '@/types/group';

// Detect topic for a chat
const chat: Chat = { /* ... */ };
const topic = detectTopic(chat);
console.log(`Topic: ${topic.topic}, confidence: ${topic.confidence}`);

// Group multiple chats
const groups = groupChatsByTopic(chats);
console.log('Groups:', Array.from(groups.keys()));

// Create a custom group
const customGroup = createCustomGroup('Mobile App Initiative', 'user123');
```
